# red-panda

A Discord bot written in JavaScript to get red pandas using a command!

## Set up

If you don't know setting up a bot application, [check this guide!](https://discordjs.guide/preparations/setting-up-a-bot-application.html).

1. Install Node.js 16.9.0 (or newer).
2. Do `npm install`
3. Replace the token using the one from your application at `index.js`.
4. Run using `node index.js`
5. Done!
